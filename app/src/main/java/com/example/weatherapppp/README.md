## Weather App

Android weather application built with Kotlin and Jetpack Compose.

### Features
- City-based weather search
- Current weather and 3-day forecast
- Offline cache support
- Unit switching (Celsius/Fahrenheit)

### Tech Stack
- Kotlin
- Jetpack Compose
- Retrofit + Gson
- Open-Meteo API
- MVVM Architecture

### API
Open-Meteo (https://open-meteo.com)
